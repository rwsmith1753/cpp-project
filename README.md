# cpp-project

University project. Used OOP to create roster from student data table, and functions for adding, modifying, and displaying data.

<strong>main.cpp:</strong> define student data. call functions to add student and print info<br>
<strong>roster.h:</strong> define roster class<br>
<strong>roster.cpp:</strong> define roster methods<br>
<strong>student.h:</strong> declare student class<br>
<strong>student.cpp:</strong> define student methods<br>
<strong>degree.h</strong> enumerate degree types
